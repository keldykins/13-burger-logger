# Foobar

Burger Logger is a Template Engine using MySQL, Node, Express, Handlebars, Bootstrap and a homemade ORM.

## Installation

Open Burger Logger folder in terminal and enter
`node server.js`

## Usage

````MySQL
Node
Express
Handlebars
Bootstrap```
